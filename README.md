# TVING 스포츠 광고 음소거

TVING 으로 스포츠 중계를 볼 때, **광고가 나오면 자동으로 소리를 끄고** 경기가 다시 시작되면 **소리를 켜주는** Chrome 확장 프로그램입니다.

광고 소리에 깜짝 놀라거나 매번 음소거 버튼을 누르는 일 없이 편하게 경기를 즐기세요. ⚾️🏀⚽️

---

## 설치 방법

처음 한 번만 하면 됩니다. (5분 정도 걸려요)

### 1. Chrome 브라우저로 확장 프로그램 관리 페이지 열기

Chrome 주소창에 아래 주소를 그대로 복사해서 붙여넣고 Enter:

```
chrome://extensions
```

### 2. "개발자 모드" 켜기

페이지 **오른쪽 위 구석**에 있는 **개발자 모드** 스위치를 켭니다.
(켜면 파란색으로 바뀝니다.)

### 3. 확장 프로그램 불러오기

왼쪽 위에 새로 나타난 **"압축해제된 확장 프로그램을 로드합니다"** 버튼을 누릅니다.

파일 선택 창이 뜨면, 이 폴더(`260523_TvingMute`)를 통째로 선택하세요.

설치 끝! Chrome 도구 모음(오른쪽 위)에 빨간색 **T** 아이콘이 나타납니다.

> 💡 아이콘이 안 보이면 도구 모음의 **퍼즐 조각 모양 버튼** 을 눌러서 확장 프로그램 목록에서 핀(📌)을 누르면 고정됩니다.

---

## 사용 방법

특별히 할 일은 없습니다. **그냥 TVING 스포츠 중계를 보면 됩니다.**

- 광고가 나오면 → 자동으로 소리가 꺼집니다 🔇
- 경기가 다시 시작되면 → 자동으로 소리가 켜집니다 🔊

### 끄고 싶을 때

Chrome 도구 모음의 **T 아이콘**을 누르면 작은 창이 뜹니다.

- **사용 스위치**: 끄면 확장이 잠시 멈춥니다. 다시 켜면 작동합니다.
- **상태 표시**: 지금 무슨 상태인지 보여줍니다.
  - 🟢 **재생 중** — 경기가 정상적으로 나오는 중
  - 🟠 **광고 감지 — 음소거됨** — 광고가 나와서 소리를 끈 상태
  - ⚫ **대기 중** — 중계 페이지가 아니거나 플레이어를 기다리는 중

### 광고 전에 이미 음소거 상태였다면?

원래 음소거 해두셨던 분이라면 **광고가 끝나도 음소거가 유지**됩니다. 마음대로 켜고 끄세요.

---

## 잘 안 될 때

### 광고인데 소리가 안 꺼져요

또는 반대로 **경기 중인데 계속 음소거** 되어 있다면, TVING 이 화면을 바꿨거나 광고 표시 방식이 변경된 것일 수 있습니다. 알려주시면 수정할게요.

임시로는 확장 아이콘을 눌러서 **사용 스위치를 끄면** 평소대로 돌아옵니다.

### 아이콘이 안 보여요

1. Chrome 도구 모음의 **퍼즐 조각 버튼** 을 누릅니다.
2. "TVING 스포츠 광고 음소거" 옆의 **핀(📌)** 아이콘을 누르세요.

### 광고 차단 프로그램(AdGuard, uBlock 등)을 쓰고 있어요

광고 차단 확장이 TVING 의 광고 표시 자체를 가리면 이 확장이 광고를 감지하지 못합니다. tving.com 에서는 광고 차단을 잠시 꺼주세요.

### 작동하던게 갑자기 안 돼요

가끔 TVING 사이트가 업데이트되면서 작동이 멈출 수 있습니다.

1. 페이지를 새로고침해 보세요 (F5).
2. 그래도 안 되면 Chrome 을 재시작해 보세요.
3. 그래도 안 되면 알려주세요.

---

## 어디서 작동하나요

`https://www.tving.com/contents/sports/.../broadcast?...` 또는 `https://www.tving.com/contents/kbo/.../broadcast?...` 처럼 생긴 **TVING 스포츠 및 KBO 야구 중계 페이지**에서만 작동합니다.

TVING 의 다른 페이지(메인, 드라마, 예능 등)에서는 아무 일도 하지 않으니 안심하세요.
